TextLine


TextLine是一款轻量级的文本编辑器。通过文字和分割线的简单组合，可以自动生成美观的 HTML/PDF 页面。


文件列表页
1.操作说明：
> cell左滑删除。
> cell右滑复制、重命名。
> cell点击进入「节点编辑页」。
> 点击底部新建按钮，创建 空白/模板 文件。



节点编辑页
1.TextLine 节点包含以下三种类型：
Br 换行
> Count 属性（1-9），表示换行数量。
Line 分割线
> Indent 属性（a-z），表示缩进级别。
Text 文本
> Indent 属性（a-z），表示缩进级别。
> Size 属性（1-9），表示文字大小。
> TextAlignment 属性（l/c/r），表示文字对齐方式。
> Bold 属性（b/f），表示文字是否加粗。
> Color 属性（#000000），表示文字颜色。

2.操作说明：
> cell左滑删除。
> cell右滑复制。
> cell点击进入「属性编辑」页。
> 点击底部新建按钮，创建新节点。
> 点击右下角按钮进入编辑模式，可进行排序、多选、复制、删除等操作。
> 点击右上角预览按钮，进入「预览页」。



预览页
1.操作说明：
> 点击右上角分享按钮，可分享 HTML 文件。
> 「设备截图 -> 选择整页 -> 完成」可分享 PDF 文件。



使用技巧
1.HTML标签中第一个属性设置为：textline='yes'，将会保留标签格式。

TextLine


TextLine is a lightweight text editor. Through the simple combination of text and dividing lines, beautiful HTML/PDF pages can be automatically generated.


File list page
1.Instructions:
cell swipe left to delete.
cell swipe right to copy and rename
cell click to enter the 'Node Edit Page'.
click the New button at the bottom to create a blank/template file.



Node Edit Page
1.TextLine Node contains the following three types:
Br(wrap)
Count attribute (1-9), indicating the number of line breaks.
Line
Indent attribute (a-z), indicating the indentation level.
Text
Indent attribute (a-z), indicating the indentation level.
Size attribute (1-9), indicating the size of the text.
TextAlignment attribute (l/c/r), indicating the alignment of the text.
Bold attribute (b/f), indicating whether the text is bold or not.
Color attribute (#000000), indicating the color of the text.

2.Instructions:
cell swipe left to delete.
cell swipe right to copy.
cell click to enter the 'Property Edit Page'.
click the New button at the bottom to create a new node.
click the button in the lower right corner to enter the editing mode, and you can perform operations such as sorting, multiple selection, copying, and deleting.
click the preview button in the upper right corner to enter the 'Preview Page'.



Preview Page
1.Instructions:
click the share button in the upper right corner can share HTML files.
'Device Screenshot -> Select Whole Page -> Finish' can share PDF files.
